# API Usage Rules

- **READ-ONLY Access Required**: Whenever you connect to the external platform APIs (such as Pancake POS), you MUST ONLY use GET requests to retrieve data.
- **NO Modifications**: Under no circumstances are you allowed to use the API key to perform POST, PUT, PATCH, DELETE, or any other state-modifying requests on the external platform. 
- The API key is strictly for pulling data into our system and must not be used to alter live data on the external shop.
